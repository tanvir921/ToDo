## Technologies Used

- Flutter (v3.7.10)
- Firebase
- REST API

## Features

- User authentication
- Add, Delete, and Modify task
- Add custom color to task
- Task finish checkbox with Line Through.
- Latest news
- Donation

## Installation

1. Clone the repository
   ```sh
   git clone https://github.com/your_username/your_repository.git
   ```
2. Install the dependencies
   ```sh
   flutter pub get
   ```
3. Run the app
   ```sh
   flutter run
   ```
